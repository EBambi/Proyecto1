Acerca de la Página

Este sitio tiene como fin ser un diagnostico de el trabajo del creador con respecto a sus conocimientos en el campo de páginas web. El sitio ha sido creado con HTML, CSS y JS (JavaScript). HTML (Hyper Text Markup Language): Es un lenguaje que se utiliza para el desarrollo de páginas web. Se usa para crear las bases del sitio, como encabezados, cuerpo, textos, etc. CSS (Cascading Style Sheets): Es un lenguaje de cómputo de diseño gráfico que sirve para definir y crear la presentación de un documento estructurado escrito en un lenguaje como HTML. Se usa para estilizar esas bases creadas con HTML. JS (JavaScript): Es un lenguaje de programación interpretado. Es orientado a objetos, basado en prototipos, imperativo y dinámico. Se usa para controlar la parte del comportamiento de diferentes elementos de la página web.
